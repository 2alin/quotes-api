# Simple Quotes API

This projects exposes a simple API that will allow users to retrieve and store quotes.

## Dependencies

We need the following:

- Node 22.21+
- SQLite 3

## Build Instructions

- Run `npm i` to install dependencies
- Run `npm run build` to setup the project
